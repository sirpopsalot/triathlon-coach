### OTC Workout Structure
All OTC workouts follow: **Warmup → Main Set → Cooldown (200 easy)**. Warmups are 600–1,000 yds and always include drills. Main sets use one of these patterns:

### Warmup Building Blocks (pick 2-3, target 400-800 yds)
- 200 easy mix & match
- 4×50 drill → swim (catchup, fist drill, scull, fingertip drag, 6-3-6, high elbows) @ Y
- 4×75 kick, swim, build @ Y
- 4×25 build 1-4 @ Y
- 8×25 hypoxic @ X
- 4×75 hypoxic @ Y

### Main Set Templates

**Endurance Ladder (1,100–2,100 yds)**
Single or double pass through descending distances with build effort.
- 4×100 build 1–4 @ :15–:20 rest
- 100 easy kick
- 4×75 build 1–4 @ :10–:15 rest
- 100 easy kick
- (Optional: 4×50 build 1–4 @ :10 rest + 50 kick)

**Broken Ladder — Up and Down (1,500 yds, 3x)**
Ascending then descending distances. Strong/80% on odds, FAST on evens.
- 25 strong → 50 FAST → 75 strong → 100 FAST → 100 strong → 75 FAST → 50 strong → 25 FAST
- :15 rest between each

**Progressive Rounds (1,400–1,650 yds)**
3 rounds, distances descend, effort increases per round.
- Round 1: 150 easy + 100 @ 85%, 150 easy + 100 @ 90%
- Round 2: 100 easy + 2×75 @ 85%, 100 easy + 2×75 @ 90%
- Round 3: 50 easy + 3×50 @ 85%, 50 easy + 3×50 @ 90%

**Cruise + Fast (800–2,100 yds)**
Long cruise swims with periodic fast efforts. Great for race-pace endurance.
- 8×100 — every 4th and 8th FAST, rest cruise @ X
- 300 pull easy
- 9×50 — every 3rd, 6th, 9th FAST, rest cruise @ X

**Descend by Round (1,400–2,400 yds, 4x)**
Same set repeated, each round faster. Track total time.
- 300 (100 FAST, 100 moderate, 100 count strokes)
- 4×50 descend 1–4, tight interval @ XX
- 100 descend by round

**Easy/Fast Ladder (750–1,500 yds)**
Ratio of easy to fast shifts each round. "Get after the 25s."
- 4×50 easy + 1×25 FAST
- 3×50 easy + 2×25 FAST
- 2×50 easy + 3×25 FAST
- 1×50 easy + 4×25 FAST
- :10 rest between all

**NS Descend (1,900 yds)**
Negative-split swims descend in distance, with IM/non-free FAST between.
- 400 NS → 100 IM FAST
- 300 NS → 100 IM FAST
- 200 NS → 100 IM FAST
- 100 strong → 100 IM FAST
- 20×25 (1ez, 1fast pattern)

### Cooldown
Always 200 yds easy — mix & match stroke, kick, etc. *Relax and get back to perfect swimming.*

### ETPA Swim Patterns (Raeleigh Harris, ETPA Racing)
Source: `~/code/tempo files/2016_ETPA_OTC_generic_70.3_training_plan.pdf` (also Olympic + Sprint versions)

ETPA plans use a **Mon / Fri / Sun** swim cadence (Mon = Mills coached or solo, Fri = pool, Sun = open water).
Swim structure is simpler than OTC coached workouts — great for solo sessions or lower-complexity days.

**Mon Mills Solo Set (ETPA 70.3 style)**
- Warmup: 100 free, 100 back, 100 kick w/ board (rest 25 for 15s), 100 pull buoy build
- Main: 3×200 @ RPE 14, focus on long DPS (distance per stroke) / :20 rest
- Main: 3×300 @ RPE 14, focus on breathing (exhale in water, breathe every 3rd stroke) / :30 rest
- Warmdown: 200 easy swim

**Fri Pool Set (ETPA, scales by plan level)**
| Plan | Set |
|------|-----|
| Sprint Wk1 | 200 WU + 4×100 relaxed effort / :30 rest + 100 WD |
| Sprint Wk2 | 300 WU + 5×100 relaxed→moderate / 1:00 rest + 100 WD |
| Olympic Wk1 | 200 WU + 10×100 relaxed effort / :20 rest + 100 WD |
| Olympic Wk2 | 300 WU + 15×100 relaxed→moderate / :20 rest + 100 WD |
| 70.3 Wk1 | 300y pull build + 10×100y alternating easy/RPE 16+ last 25 / :30 rest + 300y pull WD |
| 70.3 Wk2 | Same + add 1×300 FIRM, roll straight into WD |

**Sun Open Water (ETPA 70.3)**
- 400y warmup
- 5–10×200y moderate pace, focus on stroke length
- 10×50y building to race pace
- 200y warmdown
- Run after swim 30 min easy (neuromuscular connection)

**ETPA Kick/Drill Progressions (Mon warmup, scales by week)**
| Week | Set |
|------|-----|
| Sprint Wk1 | 4×50 flutter kick w/ board / :30 + 4×50 over-kick (slow arms, fast kick) / :30 |
| Sprint Wk2 | 4×50 flutter kick / :30 + 6×50 over-kick / :45 |
| Olympic Wk1 | 10×50 flutter kick / :30 + 10×50 over-kick / :30 |
| Olympic Wk2 | 10×50 flutter kick / :30 + 10×75 over-kick / :30 |
| 70.3 Wk1–3 | Same kick/PB build warmup, main set focus changes weekly (DPS → breathing → high elbow recovery) |

### EBTC Tri Training Day Prep (Raeleigh Harris)
Source: `~/code/tempo files/EBTC Tri Training Day Prep Plan 5 weeks.pdf`

5-week sprint prep plan. Swims are Mon, Fri, Sat (+ Sun OW if conditions allow). Uses RPE scale and "Super Sprint / Easy" intensity labels. Key patterns:

**Pool Swim (EBTC style)**
- Warmup: 4×50 free easy / :30–:40
- Main: 6–8×100 DPS or catch-up drill / :30–:40
- Kick: 4–8×25 kick with fins and board / :30
- Warmdown: 100 easy swim

**"If no swim" Alternative**
- Shoulder Shaper (21 min) — dryland shoulder/rotator cuff routine
- Nike Trainer Club app core work (10–15 min)
- Good fallback when pool access falls through

**OW Swim (EBTC, Sun)**
- Dryland warmup: arm swings, jumping jacks, 2-3 min easy run on beach
- In water: splash face, bob with forceful exhale until breath settles
- Wear wetsuit, practice getting in and out
- Keep it short — acclimatization focus, not distance

### Women's Series Training Plan (Coach Sami, Fusion Endurance)
Source: `~/code/tempo files/Womens Series training plan (1).pdf`

USA Triathlon Women's Series plan by Coach Sami (Level III USAT, Head Coach OTC). Uses **zone-based effort** (Z1–Z5) and **time-based sets** instead of yardage. Beginner-friendly progression.

**Zone Definitions**
| Zone | Description |
|------|-------------|
| Z1 | Easy, all day effort |
| Z2 | Focused but easy, can hold a conversation, sustainable 2-5 hrs |
| Z3 | Comfortably hard, one-sentence answers, sustainable 15-40 min |
| Z4 | Strong effort, 1-2 word answers, sustainable 10-20 min |
| Z5 | Hardest, no conversation, sustainable 1-3 min |

**Swim Progression (Women's Series)**
| Week | Warmup | Main Set | Cooldown |
|------|--------|----------|----------|
| 1 | 100-200 WU | 16×25 build to Swim Strong (10-15s RI) + swim 10 min (rest 5 if needed) | short CD |
| 2 | 100-300 WU + 2×50 kick/swim/kick | 100-200 into 50 fast → 50-150 into 50 fast (descending) + 8×25 strong | short CD |
| 3 | 200 WU + 4×25 drill (fins OK) + 6×50 drills (arm only, catch-up) | 3×100 perfect form + 4×50 kick w/ fins + R:20, 25 build/25 hold faster pace | 200 CD |
| 4 | 450-50 WU (swim/kick w/ board ×6) | 3×100 (75 FAST/25 easy) + 3×100 (50 FAST/50 easy) + 300 steady | 200 CD |

**Time-Based Alternative (Wk 3, Day 18)**
- 3×7 min swim (steady, effortless). Rest 2 min between.
- Then 6-8×50 build each 50. :15 RI.
- Good for days when counting laps feels like too much.

### Scaling Guidelines
- **~1,700 yds**: 400 warmup + 1,100 main (1 pass through ladder or 2x easy/fast) + 200 cooldown
- **~2,200 yds**: 600 warmup + 1,400 main + 200 cooldown
- **~2,800 yds**: 800 warmup + 1,800 main + 200 cooldown
- **~3,500 yds**: 800 warmup + 2,400 main (full OTC volume) + 200 cooldown + extra pull

### When to Use Which Style
| Mood / Context | Style | Why |
|----------------|-------|-----|
| Coached Monday at Mills | OTC format | Coach runs structured intervals, effort columns |
| Solo midweek, feeling motivated | OTC main set template | Structured, variety, keeps you honest |
| Solo midweek, want simple | ETPA Fri pool set | Just 10×100 relaxed. Done. |
| Low energy, just get in the water | Women's Series time-based | Swim 7 min ×3. No counting. |
| Open water practice | ETPA Sun OW | 200s at moderate + 50s at race pace |
| Can't get to pool | EBTC dryland alternative | Shoulder Shaper + core |
